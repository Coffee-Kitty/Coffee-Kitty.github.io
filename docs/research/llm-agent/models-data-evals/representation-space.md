## 最小二乘、 词向量、llm
经典的词向量例子：国王 - 男人 + 女人 ≈ 王后。

在 LLM 的深层表示中，类似的关系更加丰富：柏林 - 德国 + 法国 ≈ 巴黎，甚至(动词的过去式) - (动词原形) 在空间中指向相近的方向。

这意味着，LLM 找到的规则，最终让原本纠缠在一起的原始文本序列，经过层层非线性扭曲，落到了一个结构极其清晰的几何空间。在这个空间里，类比、推理、情感倾向等都可以用向量的线性操作来近似——就像你最初说的“矩阵是空间的扭曲”，LLM 把这个逻辑推到了极致。
