### [3419. 图的最大边权的最小值](https://leetcode.cn/problems/minimize-the-maximum-edge-weight-of-graph/)

关键词：二分答案、反向建图、DFS 连通性检查。

```python
class Solution:
    def minMaxWeight(self, n: int, edges: List[List[int]], threshold: int) -> int:
        """
        1. 将所有边反向，题目要求变为从 0 节点可以到达其它所有节点。
        2. 要让最大边权尽可能小，可以二分最大允许边权 upper。
        3. 在 upper 限制下做 DFS，检查是否能访问所有点。
        """
        g = [[] for _ in range(n)]
        for a, b, w in edges:
            g[b].append((a, w))

        vis = [0] * n

        def check(upper) -> bool:
            def dfs(i) -> int:
                cnt = 1
                vis[i] = upper
                for ni, wi in g[i]:
                    if vis[ni] != upper and wi <= upper:
                        cnt += dfs(ni)
                return cnt

            return dfs(0) == n

        maxw = max(e[2] for e in edges)
        idx = bisect_left(range(1, maxw + 1), True, key=check)
        return -1 if idx >= maxw else idx + 1
```
