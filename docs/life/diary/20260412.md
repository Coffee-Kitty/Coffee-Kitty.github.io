

## 简单学习下dask

Dask 的核心是构建一张任务依赖图（DAG），
每个节点表示一个计算任务（function + arguments），
边表示数据依赖关系。

Dask 对象（delayed / bag / array / dataframe）都是这张图的不同抽象层。

计算是 lazy 的，只有在调用 compute 时才会执行，
调度器根据 DAG 自动并行执行可独立的任务。


Dask = DAG + Lazy + Scheduler
