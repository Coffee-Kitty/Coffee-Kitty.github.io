## 写作模板

一篇学习笔记可以尽量回答四个问题：

```text
1. 这件事解决什么问题？
2. 它的核心机制是什么？
3. 我实际复现或使用时遇到了什么？
4. 这件事对我下一步有什么影响？
```

一篇日记可以尽量保留三个信息：

```text
今日推进：
主要问题：
下一步：
```
